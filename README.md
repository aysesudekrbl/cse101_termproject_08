Fitness Tracking App

-This is a simple command-line fitness tracking application developed for a CSE 101 course.

-The program is run using python main.py.

-Users can register with an email and PIN and log in to the system.

-Logged-in users can add, update, and delete workouts, meals, and health metrics.

-Data is stored in JSON files and saved automatically when the program exits.

-The application can generate basic terminal reports such as workout summaries, calorie totals, macro breakdowns, and metric summaries.
